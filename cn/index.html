Vantage学生帮


我们是一个高中生成立的辅导组织，来自于大多伦多 UTS，UCC，和 Bayview SS 等顶级公私立高中 11 年级以上的学生。我们是不以营利为目的，希望能帮到学弟学妹们，引导他们对学习产生兴趣，而与此同时我们也要用辅导的盈利回馈社会，帮助有需要的人。

教师简介

我们的团队有 70 多个导师。导师们都是多伦多不同的 program 和高中的学生。



Andrew Shi

Andrew 是学生帮的辩论和公众演讲教师。他赢得了 20 多个国际、国家和省级的辩论比赛。 他有超过 8 年的演讲经验，也有辅导小学生的经验。去年，他赢得了加拿大国家级辩论赛和萨斯卡通的世界学校锦标赛 (Saskatoon World School Tournament)

Vanessa Poiana

Vanessa 是学生帮中的数学和科学教师。她在校内与校外中的数学、逻辑、写作与科学竞赛中都获得了名次。同时在 DECA 竞赛和 Mock Trial 辩论赛中取得一等奖。她还被约翰霍普金斯大学 CYT Program 和杜克大学 TIP program 录取。

Cheryl Geng

Cheryl 是学生帮的绘画导师。她已经有超过 8 年的绘画经验，和多种绘画艺术风格：数码、水彩、丙烯、超现实主义、油画。她拥有丰富的艺术技巧知识和教学能力。

Jayson Tian

Jayson 是学生帮的编程和英文教师。他在许多编程、数学、英文写作、中文和生物竞赛中取得了名次。他的业余爱好是绘画，漫画制作，音乐制作，而用 C# 和 Javascript 编程游戏。今年，他会代表加拿大参加国际奥林匹克生物竞赛。

Patrick Nova

Patrick 是学生帮中的法语教师之一. 他有超过 11 年的法语交流与学习经验。他去年在法国的波尔多留学与度假了三个月，帮助了他大大提高他对法语的流利度而让他学到了很多法国的文化。除了法语，他还会英文、中文、西班牙语、和罗马尼亚语。
